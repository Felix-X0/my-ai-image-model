# my-ai-image-model
# Anime Dark Fantasy Generator
Model AI ini dirancang untuk menghasilkan karakter dengan gaya dark fantasy, katanas, dan nuansa atmosferik.

## Deskripsi
Model ini berfokus pada estetika karakter anime dengan detail tinggi, pencahayaan dramatis, dan gaya artistik yang edgy.

## Cara Penggunaan
Masukkan prompt seperti: "Dark fantasy anime character, white hair, red eyes, katana, dark armor, snowy background".
